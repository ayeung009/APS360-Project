# A Deep Learning Assistant for Digital Circuit Design

*Digital circuits are a fundamental building block of modern computing, enabling technologies such as
CPUs, FPGAs, and ASICs that power applications ranging from artificial intelligence to robotics. At
their core, digital circuits are composed of logic gates that perform Boolean operations such as AND,
OR, and NOT, and can be interconnected to form increasingly complex functions. The goal of
this project is to derive the equivalent Boolean expression from an image or sketch of a digital circuit
composed of interconnected logic gates.*

## 🚀 Getting Started

### Data

All data used for training, testing, and validation is stored in the ```data``` folder. Check out the dataset directory for more information [here](https://github.com/ayeung009/APS360-Project/tree/main/data). Data processing is done in ```Data_Processing.ipynb```.

### Models

Model training and setup is stored in the ```models``` folder. Namely, see ```Vision_Transformer_Baseline.ipynb``` and ```CNN_Primary.ipynb``` for the models.
